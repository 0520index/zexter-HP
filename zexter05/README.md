# 株式会社zexter 公式ホームページ

白×金を基調にした、動きのあるコーポレートサイトです。

## ページ構成

| ファイル | 内容 |
|---------|------|
| `index.html` | トップ（イントロ、ヒーロー、事業軌道） |
| `about.html` | 会社情報 |
| `services.html` | 事業内容（開閉式） |
| `contact.html` | お問い合わせ |

## 見方（初心者向け）

1. フォルダ内の **`index.html`** をダブルクリックするとブラウザで開けます
2. 上のメニューから各ページに移動できます
3. 住所・電話・メールは仮の文言なので、確定後に書き換えてください

## 主なギミック

- 金箔パーティクルが漂う背景
- 金色のページ切り替えカーテン
- カスタムカーソル / マグネットボタン
- トップの事業「軌道」インタラクション
- スクロール進捗バー・出現アニメーション

## 本番に向けて差し替える場所

- 会社概要の所在地・設立日（`about.html`）
- 電話・メール・営業時間（`contact.html`）
- お問い合わせフォームの送信先（現在はデモ）
