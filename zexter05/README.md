## 🚀 使用方法

### 1. ファイルのダウンロード
このフォルダ内の全てのファイルをダウンロードしてください。

### 2. ブラウザで開く
`index.html`をダブルクリックするか、ブラウザにドラッグ&ドロップしてください。

### 3. ローカルサーバーで開く（推奨）
より良いパフォーマンスのため、ローカルサーバーでの実行を推奨します：

**VS Code Live Server拡張機能を使用する場合:**
1. VS Codeで`index.html`を開く
2. 右クリックして「Open with Live Server」を選択

**Pythonを使用する場合:**
```bash
# プロジェクトフォルダで実行
python -m http.server 8000
# ブラウザで http://localhost:8000 にアクセス
```

**Node.jsを使用する場合:**
```bash
# http-serverをインストール
npm install -g http-server
# プロジェクトフォルダで実行
http-server
```

## 🎯 カスタマイズ方法

### 色の変更
`styles.css`内の以下の値を変更してください：
```css
/* メインカラー */
#ffd700  /* 金色 */
#ffffff  /* 白色 */
#000000  /* 背景色（黒） */
```

### 会社情報の変更
`index.html`内の以下のセクションを編集してください：
- 会社名: `ZEXTER`
- 電話番号: `000-0000-0000`
- メールアドレス: `info@zexter.co.jp`
- 住所: `東京都○○区○○`

### サービス内容の追加・変更
`index.html`の`services-grid`セクションでサービスカードを追加・編集できます。

### アニメーションの調整
`script.js`内のタイミングや効果を調整できます：
```javascript
// ローディング時間の変更
setTimeout(() => {
    loadingScreen.classList.add('hidden');
}, 2000); // この値を変更（ミリ秒）
```
