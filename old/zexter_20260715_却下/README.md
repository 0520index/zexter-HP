# 株式会社ZEXTER コーポレートサイト

## プロジェクト概要

株式会社ZEXTERの公式コーポレートサイト。近未来的なデザインと高級感を兼ね備えた、ブランド価値を高めるWebサイトです。

## デザインコンセプト

- **近未来**: 最新技術を感じさせるモダンなUI/UX
- **高級感**: BMWのような洗練されたデザイン
- **ゲーミング要素**: Riot Gamesのような迫力ある演出
- **ミニマル**: Appleのような美しい余白設計

## カラーパレット

- **メイン**: 白ベース（#FFFFFF）
- **アクセント**: 金色（#D4AF37）
- **サブ**: 黒（最小限使用）

## 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなスタイリング
- **JavaScript**: インタラクション制御
- **GSAP**: 高性能アニメーション
- **Lenis**: スムーススクロール
- **Splitting.js**: テキストアニメーション
- **Three.js**: 3D要素

## フォルダ構成

```
zexter/
├── index.html                 # メインHTML
├── css/                      # スタイルシート
│   ├── reset.css            # CSS リセット
│   ├── design-system.css    # デザインシステム
│   ├── components.css       # コンポーネント
│   ├── layout.css          # レイアウト
│   ├── sections.css        # セクション別
│   └── animations.css      # アニメーション
├── js/                      # JavaScript
│   ├── main.js             # メインスクリプト
│   ├── modules/            # 機能別モジュール
│   └── utils/              # ユーティリティ
├── assets/                  # アセット
│   ├── images/             # 画像（WebP推奨）
│   ├── icons/              # アイコン
│   └── fonts/              # フォント
├── components/              # コンポーネント
│   └── ui/                 # UIコンポーネント
└── utils/                   # 共通ユーティリティ
```

## パフォーマンス目標

- **Lighthouse**: 95点以上
- **FPS**: 60fps維持
- **アクセシビリティ**: WCAG 2.1 AA準拠
- **SEO**: 最適化済み

## セクション構成

1. **Hero** - インパクトのあるファーストビュー
2. **About** - 会社概要・理念
3. **Business** - 事業内容
4. **Strength** - 強み・特徴
5. **Company** - 会社情報
6. **Contact** - お問い合わせ

- コンポーネント化を意識
- 保守しやすい設計
- レスポンシブ対応必須
- SEOに強いHTML構造
- アクセシビリティ対応
- 軽量なアニメーション
